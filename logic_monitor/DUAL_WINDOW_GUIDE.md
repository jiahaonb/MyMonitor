# 双窗口系统实现说明

## ✨ 新功能

你的应用现在有**两个独立的窗口**：

### 1. 快捷亮度面板（QuickPanel）
- **触发方式**：左键点击托盘图标
- **功能**：仿 macOS 控制中心，只显示亮度滑块
- **尺寸**：320x200 像素，紧凑设计
- **特点**：
  - 点击外部自动隐藏
  - 始终在最前显示
  - 完全模仿 macOS 亮度调节样式
  - Windows/macOS 自动适配位置

### 2. 完整设置窗口（Settings）
- **触发方式**：右键托盘图标 → 选择"设置"
- **功能**：完整的亮度设置界面（你之前做的界面）
- **尺寸**：420x650 像素
- **特点**：
  - 预设值按钮网格
  - 精确控制滑块
  - 多显示器支持

## 📂 新增文件

1. **QuickPanel.vue** - 快捷面板组件
   - 路径：`src/renderer/src/QuickPanel.vue`
   - 简洁的 macOS 风格滑块界面

2. **quick.html** - 快捷面板 HTML
   - 路径：`src/renderer/quick.html`

3. **quick.js** - 快捷面板入口
   - 路径：`src/renderer/quick.js`

## 🔧 修改的文件

### src/main/index.js
- 重构了窗口创建逻辑
- 实现双窗口系统
- 优化托盘交互：
  - 左键 → 快捷面板
  - 右键 → 菜单（设置/退出）
- macOS/Windows 位置自适应

## 💡 平台差异处理

### macOS
- 托盘在顶部菜单栏
- 快捷面板从图标下方弹出
- 使用 `vibrancy: 'popover'` 效果

### Windows
- 托盘在底部任务栏
- 快捷面板从图标上方弹出
- 支持 Windows 11 圆角

## 🚀 使用方法

1. **日常使用**：左键点击托盘 → 快速调整亮度
2. **高级设置**：右键托盘 → 设置 → 详细配置
3. **退出应用**：右键托盘 → 退出

## 🎨 设计特点

快捷面板完全模仿 macOS 控制中心：
