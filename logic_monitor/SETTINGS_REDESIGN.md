## ⚙️ 配置系统

### 默认配置
```javascript
{
  panelOpacity: 0.85,  // 面板透明度
  language: 'zh-CN',   // 语言
  version: '1.0'       // 版本号
}
```

### API
```javascript
import { configManager } from './utils/config.js'

// 获取配置
configManager.get('panelOpacity')

// 设置配置
configManager.set('panelOpacity', 0.7)

// 获取所有配置
configManager.getAll()
```

## 🚀 使用方法

1. **调节亮度**：点击"亮度"标签 → 选择预设值或使用滑块
2. **调节透明度**：点击"配置"标签 → 拖动滑块调整
3. **查看版本**：配置页面底部显示
